Hexo 搭建

2019-07-19 工具 0 评论

1、准备工作

在github上创建一个仓库，命名规则： github账户名 + .github.io

2、全局安装hexo

npm install hexo -g

3、新建文件夹，初始化

hexo init
npm install

4、配置config.yml

必配deploy：
    type：git
    repo: https://github.com/yourSite/yourSite.github.io
    branch: master

5、创建页面

hexo new pageTitle

6、开启本地服务

hexo clean  清除缓存
hexo generate  生成静态页面
hexo server 启动本地服务

7、部署到git服务器

hexo deploy
